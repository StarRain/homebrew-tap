# StarRain Homebrew Tap

Homebrew formulas for StarRain projects.

## Codex Dream Skin Manager

Install and start the local macOS service:

```bash
brew install starrain/tap/codex-dream-skin-manager
brew services start codex-dream-skin-manager
```

Open <http://localhost:19341/management.html> after the service starts.

Project documentation: <https://github.com/StarRain/Codex-Dream-Skin-Manager>
